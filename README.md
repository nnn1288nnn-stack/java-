## Usage Instructions

### Prerequisites
- Java Development Kit (JDK) 11 or higher
- Compatible IDE (IntelliJ IDEA, Visual Studio Code, Eclipse)

### Running the Code

Execute the main driver:
```bash
javac javapractice/Main.java
java javapractice.Main
```

### Compiling Individual Modules

```bash
javac homework1_2.java
java homework1_2

javac homework1_3.java
java homework1_3
```
